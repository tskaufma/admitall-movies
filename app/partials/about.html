About AdmitAll

AdmitAll is an AngularJS application to view the backup file created by the AdmitOne Movie Catalog app for Android Devices. The export file is read and displayed to allow filtering and sorting of the movies.

This product uses the TMDb API but is not endorsed or certified by TMDb.

The source code for this application is available on GitHub at https://github.com/tskaufma/admitall-movies
